# soil-test-data-auto-process
土工试验应力应变数据自动化处理Python脚本

## 项目功能
1. 批量遍历文件夹，批量读取试验csv数据文件
2. 自动计算黏聚力、内摩擦角、峰值偏应力等土力学强度指标，剔除异常试样
3. 批量绘制应力-应变曲线图，自动导出标准化汇总报告

## 技术依赖
pandas、numpy、matplotlib、os文件批量处理

## 项目成果
原人工处理单批次试验数据需2小时，脚本自动化仅需1分钟，支持上百组试样批量计算，大幅降低人工重复工作量。# soil-test-data-auto-process
